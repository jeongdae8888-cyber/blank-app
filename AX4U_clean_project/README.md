# AX4U 교통사고 영상 분석기

AX4U는 블랙박스 또는 CCTV 영상에서 사용자가 직접 지정한 프레임과 실제 이동거리를 기준으로 속도, 공주거리, 제동거리, 정지거리, 회피각도, 사고 회피 가능성을 계산하는 Windows용 보조 프로그램입니다.

## 실행

```powershell
pip install -r requirements.txt
$env:PYTHONPATH="src"
python -m ax4u.main
```

## 핵심 원칙

- 속도는 `실제 이동거리 / 실제 경과시간 * 3.6`으로 계산합니다.
- 실제 이동거리가 없으면 임의 속도를 만들지 않고 `검증불가`와 경고를 표시합니다.
- 공주시간 입력값과 영상의 공주-제동 프레임 시간은 별도로 표시합니다.
- 공주 시점과 제동 시점이 같은 프레임이어도 오류로 처리하지 않습니다.
- 산출값이 비정상적으로 커도 숨기지 않고 경고와 계산 근거를 함께 표시합니다.

## 주요 기능

- 영상 열기 및 1프레임 단위 이동
- 시작, 공주, 제동, 회피 시작, 회피 끝, 접촉, 속도 시작, 속도 끝 시점 지정
- 지정 버튼의 눌림 상태 유지 및 개별/전체 초기화
- 실제 이동거리, FPS, 공주시간, 마찰계수, 경사, 가용거리 입력
- 계산 근거와 진단값 표시
- 보고서 복사, CSV 저장, JSON 프로젝트 저장/불러오기
